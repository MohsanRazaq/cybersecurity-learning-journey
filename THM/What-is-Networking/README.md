# What is Networking?

## Platform
TryHackMe

## Objective
Understand basic networking concepts and communication between systems.

## Topics Learned
- IP addresses
- Ports
- Routers
- Switches
- DNS
- HTTP/HTTPS
- Packets

## Concepts Understood
- Devices communicate through protocols.
- Ports identify services running on systems.
- DNS translates domain names into IP addresses.

## Key Takeaways
Networking knowledge is critical for both offensive and defensive security.

## Security Perspective
Misconfigured services and exposed ports can create attack surfaces.
